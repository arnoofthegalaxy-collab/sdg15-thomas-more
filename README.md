# SDG 15 Website - Thomas More Theme

Deze Streamlit app focust op **SDG 15: Life on Land** en is gebouwd met een Thomas More look and feel.

## Wat doet de app?

- Biodiversiteitsscan met gepersonaliseerde tips
- Actieplanner met impactscore per activiteit
- Natuurdagboek met CSV-export
- Community pledges

## Lokaal starten

1. Installeer dependencies:

```bash
pip install -r requirements.txt
```

2. Start de app:

```bash
streamlit run app.py
```

## Publiceren

- Zet deze map op GitHub
- Koppel de repo aan Streamlit Community Cloud
- Kies `app.py` als startbestand
